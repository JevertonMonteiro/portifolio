Meus Projetos

HDC Host

HC Host é um projeto estático desenvolvido em HTML e CSS, criado para apresentar servidores de hospedagem de forma clara e profissional, com foco em design limpo e estrutura organizada.

Academia Pick Forte

Academia Pick Forte é um projeto desenvolvido em HTML, CSS e JavaScript, criado para apresentar os serviços de uma academia moderna. Ele possui design atraente e um sistema de cadastro que envia automaticamente os dados dos alunos para uma planilha Excel.

Clone Netflix

Clone Netflix é um projeto estático desenvolvido em HTML e CSS, criado para reproduzir o visual da Netflix com layout moderno. Ele demonstra minhas habilidades em front-end e design, mesmo sem funcionalidades dinâmicas ou adaptação para dispositivos móveis.
